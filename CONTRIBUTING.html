HOW TO CONTRIBUTE

여러 가지 방법으로 해당 Repository에 참여하실 수 있습니다 :) PR을 올려주실 때, labels를 참고하셔서 알맞은 제목을 함께 올려주세요! Commit Message는 Update라고만 해주셔도 되고, 원하시는 메세지를 적어주시면 됩니다.

Labels for PR

  • Edit typos or links
    오타 또는 잘못된 링크를 수정.
  • Inaccurate information
    잘못된 정보에 대한 Fix.
  • New Resources
    새로운 자료 추가

그 외 Labels

  • Suggestions
    해당 Repository에 건의하고 싶은 사항에 대해서 Issue로 등록해주세요.
  • Questions
    질문이 있으시면 해당 Label과 함께 Issue를 등록해주세요.
